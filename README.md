# Personal Website

Simple static personal website with sections for:

- About me
- Previous experience
- Planned future work
- Contact information

## Run locally

From this folder:

```bash
python3 -m http.server 8000
```

Then open: <http://localhost:8000>

## Customize

Edit `index.html` to update:

- Name and headline
- Experience cards
- Future work list
- Email and LinkedIn links

Edit `styles.css` to change colors, spacing, and fonts.
